        Siempre que empecemos a trabajar con proyectos que ya han sido modificados y que tienen dependencias definidas, debemos instalar las dependencias declaradas en el archivo package.json utilizando el siguiente comando:
        > npm install
        Instalará las dependencias dentro de node_modules.

        Para utilizar las dependencias externas instaladas se utiliza require() y el nombre de la dependencia:

        const cowsay = require('cowsay')

console.log(cowsay.say({
    text: 'Hello World!',
    e: '-O',
    T: 'U'
}))
